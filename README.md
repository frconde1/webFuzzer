# webFuzzer
Cybersecurity tool for fuzzing web pages. Coded in Python

## Usage
```
python3 script.py <target url> 
```
## Optional parameters
```-w --wordlist```: Provide a wordlist instead of the default one \

```-t --threads```: Set thread quantity (Default = 10) \

```--delay```: Delay between queries (No delay by default) \
